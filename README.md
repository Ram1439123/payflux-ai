# PayFlux AI

### Autonomous Stablecoin Payroll & Cross-Border Settlement Infrastructure on Arc

PayFlux AI is an AI-powered stablecoin payment infrastructure built on Arc that enables businesses to automate global payroll, contractor payouts, and cross-border settlements using USDC.

Built for the Stablecoin Commerce Stack Challenge.


# 🌍 Overview

PayFlux AI combines:

* AI agents
* programmable payment workflows
* stablecoin settlement infrastructure
* cross-chain payment routing

to create autonomous financial workflows for modern global businesses.

The platform allows businesses to:

* automate payroll
* manage contractor payouts
* route treasury flows
* settle cross-border payments
* generate real-time payment receipts

using Circle infrastructure on Arc.

# ⚙️ Setup Guide

## Clone Repository

```bash
git clone https://github.com/Ram1439123/payflux-ai.git
```

## Install Dependencies

```bash
npm install
```

## Start Frontend

```bash
cd frontend
npm run dev
```

## Start Backend

```bash
cd backend
npm start
```
