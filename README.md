 PayFlux AI

AI-powered stablecoin payroll & cross-border settlement infrastructure built on Arc using Circle developer tools.

Overview

PayFlux AI enables businesses to automate:

* global payroll
* contractor payouts
* treasury routing
* cross-chain stablecoin settlement

using USDC and programmable payment workflows.

Built for the Stablecoin Commerce Stack Challenge.



 Tracks

* Best Agentic Economy Experience on Arc
* Cross-Border Payments & Remittances



Circle Products Used

* USDC
* Circle Wallets
* Circle Gateway
* CCTP + Bridge Kit
* Nanopayments


Features

* AI payroll automation
* Embedded wallet onboarding
* Cross-chain settlement
* Treasury routing
* Real-time payment receipts



Tech Stack

Frontend:

* Next.js
* TailwindCSS

Backend:

* Node.js
* Express.js

Blockchain:

* Arc Smart Contracts
* Circle APIs


MVP Workflow

1. Upload payroll batch
2. AI validates payments
3. Treasury routes funds
4. USDC settles on Arc
5. Contractors receive payment
6. Dashboard generates receipts
